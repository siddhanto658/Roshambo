## ✨ Features

- **P2P Multiplayer** — Direct using WebRTC
- **Real-time Battle peer-to-peer connection** — Instant move synchronization
- **AI Announcer** — Hype commentary for every match
- **Score Tracking** — Track wins across rounds
- **Responsive Design** — Works on desktop and mobile

---

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 🎮 How to Play

1. **Host a Game** — Copy your Room ID and share it with a friend
2. **Join a Game** — Enter your friend's Room ID to connect
3. **Battle** — Both players select their move (rock, paper, or scissors)
4. **Reveal** — Moves are revealed simultaneously
5. **Rematch** — Play again!
